## 行程
```
札幌JR車站(北口) ->北海道大學 --> 白色戀人公園 --> JR札幌駅 --> JR函館駅 --> 五稜郭塔
```
## 交通
- 北海道大學
```
札幌JR站，從北口行約五分鐘便到北海道大學的入口。
```
- 白色戀人公園
```
坐地鐵東西線(橘線: T01)到宮之澤站的5號出口徒步7分
```
- 五稜郭公園(五稜郭塔) : 函館市五稜郭町43-9
```
1. 在函館駅前搭路面電車,在五稜郭公園前站下車,走路15分鐘
2. 函館巴士　“五稜郭公園入口” 下車，走路7分鐘
```
